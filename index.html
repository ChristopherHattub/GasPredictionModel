About the Project

This project was developed for a client in the gas and oil industry. It leverages machine learning techniques to predict customer spending behavior in the gas and oil industry. By combining real historical customer usgedata with local weather conditions, the model provides accurate consumption forecasts to support better business decision-making.

Built with Python, Pandas, NumPy, and Scikit-learn, the model was deployed using scalable Microsoft Azure pipelines, ensuring performance, reliability, and adaptability across varying data volumes. Developed comprehensive architecturea design and deployment pipeline.
